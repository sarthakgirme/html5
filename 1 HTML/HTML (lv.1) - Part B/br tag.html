Bold, Italic, Underline Tags.

Used to highlight the Text in your Page.

  • b Tag. (Bold)

    Used to bold the text.

    Example : Tiger, Lion.

  • i Tag. (Italic)

    Used to make italic text.

    Example : Tiger, Lion.

  • u Tag. (Underline)

    Used to underline the text.

    Example : Tiger, Lion.
