Anchor Element.

it is used to add Links to your Page.

Basically if we click on any word, then we will redirect on any other Page or Section.

We uses a Tag for Anchor / Linking.

we'll have to write href="" Attribute after a in openning tag.

  • href stands for Hypertext Refference.
  • It is just Linking.
  • in href="", we'll have to Write complete Link in " ".
  • in between openning and closing tags we will write our Text, so when we'll click on that Text we will Redirect to link specified in href="".
  • e.g. Google .
    • I've written the Attribute, href="https://www.google.com/".

There are two types of links we can write in href="".

  1. Absolute Link. (Internet) :
    • As it is Online Website link.
  2. Relative Link. (File System) :
    • We give the Directory Link from our Device.
