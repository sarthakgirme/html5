Image Element in HTML.

Used to Add Image to your Page

Previously we've seen Pair tags like p tag, a tag, h tags.

  • It consist of Openning tag and Clossing tag.

Image tag (img tag) is Single tag.

  • It has only one (single) tag.
  • It has 2 Attributes.
    1. First Attribute is : src="". image source link.
    2. src link can be Absolute URL or Relative Link.
    3. Second Attribute is : alt="". alternate.
    4. In alt="", We write alternate Text about the image. If any reason image link not works, Page will Show alternate text instead of Image.
  • We can add 2 more Attributes : height="" and width="".
  • But we'll arrange image size using CSS.
