Action Attribue.

- Action attribute is used to define what action needs to be performed when a form is submitted or where the form data should be sent.
- After filling all the data ijnto the form and after submitting the form what need to do with all this data is specified by this Action Attribute.
- Action attribute used to tell that, sent all the data at address specified in action attribute.

Example :

1) <form action="/action.php">
2) <form actio="/action">
