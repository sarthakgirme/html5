Tables in HTML

Combination of Rows and Columns | Tables are used to Represent real life table data.

Row :

If analyzing the table Horizontally.

Column :

If analyzing the table Vertically.

Table Header

Main Row (or can say main Content) in the Table.

Components of Table :

  • (<table>)     | Table.
  • (<caption>) | Caption of Table.
  • (<tr>)          | Table Row. (To create/display Individual Row.)
  • (<th>)          | Table Header. (Used to display Header.)
  • (<td>)          | Table Data. (Used to display Data.)
